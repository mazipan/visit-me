## Install

```
$ npm i -g visit-me
```
## Usage

```
$ visit-me url=[YOUR_WEB_URL]
```
## Additional Options

| Argument    | Alias  | Description                    | Data Type | Default Value |
| ----------- | ------ | ------------------------------ | --------- | ------------- |
| `--show-ui` | `--ui` | Show the browser when visiting | `boolean` | `false`       |
| `--count`   | `--c`  | Visit count                    | `number`  | `1`           |
